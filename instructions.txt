Create your first React App!

1. In this exercise you are going to create your own book store by using React Components!
2. Use the create-react-app boilerplate to create your new React application
3. Install and set up bootstrap + react-bootstrap in your app
4. Create a MyNav component which will render a bootstrap navigation bar with the following links: Home, About and Browse. Note: we don't know how  to perform navigation yet, so for now just put "#" as the href for each one on them
5. Create a MyFooter component which will render a footer for the page
6. Create a Welcome component with a Jumbotron and a subtitle for your shop
7. Create a Latest Release component. This component should read from one of the json files and print out the cover of the books
8. Push everything on GitHub and publish on Eduflow before 17:00 CET
